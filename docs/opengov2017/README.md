# OpenGov 2017 Reference Docs

These notes are verbose extraction-style reference documents based on the two key Government of Canada publications tied to the 2017-2018 open government consultation cycle:

1. [01-what-we-heard-summary.md](./01-what-we-heard-summary.md)
2. [02-2018-2020-national-action-plan.md](./02-2018-2020-national-action-plan.md)
3. [codex-gold-standard-qa.md](./codex-gold-standard-qa.md)

Purpose:

- preserve a readable benchmark record of what the government itself said
- support comparison between Broadly-generated reports and the government’s own framing
- separate the consultation summary from the government’s action-oriented response

These are not verbatim transcripts. They are structured extraction notes intended for analysis and benchmarking.
